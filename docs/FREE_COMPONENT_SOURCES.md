# 10 Trang Component Free (Marquee / Logo Cloud & UI)

Danh sách 10 nguồn component miễn phí có Logo Marquee, Logo Cloud hoặc UI tương tự:

---

## 1. **Aceternity UI** – Logo Clouds & Marquee
- **URL:** https://ui.aceternity.com
- **Logo Cloud:** https://ui.aceternity.com/blocks/logo-clouds
- **Infinite Moving Cards:** https://ui.aceternity.com/components/infinite-moving-cards
- Copy-paste, Tailwind + Framer Motion, 200+ component

## 2. **shadcn/ui** – Marquee
- **URL:** https://ui.shadcn.com
- **Marquee:** `npx shadcn@latest add https://www.shadcn.io/r/marquee`
- Radix UI, Tailwind, tích hợp dễ với Next.js

## 3. **react-fast-marquee** (npm)
- **URL:** https://www.react-fast-marquee.com
- **npm:** `npm i react-fast-marquee`
- Marquee nhẹ, MIT, ~349K downloads/tuần

## 4. **Tailwind Components**
- **URL:** https://tailwindcomponents.com
- 600+ mẫu cộng đồng, copy-paste

## 5. **Sera UI**
- **URL:** https://seraui.seraprogrammer.com
- Marquee React với speed, direction, pause on hover

## 6. **Zippy Starter**
- **URL:** https://zippystarter.com
- LogoMarquee: https://zippystarter.com/docs/components/logo-marquee

## 7. **Cautron Marquee** (GitHub)
- **URL:** https://github.com/cautronsoftware/cautron-marquee-logo-widget
- Splide.js, MIT, logo marquee responsive

## 8. **Tailkits / SyntaxUI**
- **URL:** https://tailkits.com
- 100+ component React + Tailwind, có thể có marquee

## 9. **Magic UI**
- **URL:** https://magicui.design
- Component animated, có thể có logo strip

## 10. **Radix UI** (nguyên liệu base)
- **URL:** https://www.radix-ui.com
- Primitives để tự build marquee/carousel

---

## Dùng trong project này

Logo Marquee hiện dùng **react-fast-marquee** – được gợi ý bởi Aceternity Logo Clouds.
